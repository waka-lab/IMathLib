function

iml::sort

概要

iml::sortはイテレータ範囲を任意の値で埋める。
また、戻り値はコピーされた最後のイテレータ位置の次の位置である。 


namespace iml {
	template <class Iterator, class Predicate = iml::type_comparison<typename iml::iterator_traits<Iterator>::value_type>>
	void sort(Iterator first, Iterator last);
}

要求

Iteratorはランダムアクセスイテレータか双方向イテレータ、前方向イテレータのいずれかでなければならない。
また、Predicateは小なり比較を行う静的メンバ関数ltが定義されていなければならない。

動作

Iteratorはランダムアクセスイテレータか双方向イテレータならばイントロソートを行い、前方向イテレータならばコームソートを行う。

サンプルコード

入力


#include <IMathLib/IMathLib.hpp>

//逆順ソートのための構造体
template <class T>
struct pred {
	static constexpr bool lt(const T& v1, const T& v2) { return v1 > v2; }
};

int main() {

	size_t temp[10];
	iml::rnd::Xorshift32_32<size_t> rnd_xorshift(123);
	
	//適当に整数値を生成
	for (size_t i = 0; i < 10; ++i) {
		temp[i] = rnd_xorshift.get();
		std::cout << i << "_th:" << temp[i] << std::endl;
	}
	
	iml::sort(temp, &temp[10]);
	std::cout << "\nsorted" << std::endl;
	for (size_t i = 0; i < 10; ++i)
		std::cout << i << "_th:" << temp[i] << std::endl;
	iml::sort<size_t*, pred<size_t>>(temp, &temp[10]);
	std::cout << "\nreverse sorted" << std::endl;
	for (size_t i = 0; i < 10; ++i)
		std::cout << i << "_th:" << temp[i] << std::endl;
		
	return 0;
}
