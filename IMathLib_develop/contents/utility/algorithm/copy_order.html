function

iml::copy_order

概要

iml::copy_orderはイテレータの始点から別のイテレータ範囲分をコピーする。
iml::copy_reverse_orderとは対称的な関数である。
また、戻り値はコピーされた最後のイテレータ位置の次の位置である。 


namespace iml {
	template <class OutputIterator, class InputIterator>
	OutputIterator copy_order(OutputIterator first1, InputIterator first2, iml::imsize_t n);
	template <class OutputIterator, class InputIterator>
	OutputIterator copy_order(OutputIterator first1, InputIterator first2, InputIterator last2);
}

サンプルコード

入力


#include <IMathLib/IMathLib.hpp>

int main() {

	int temp[10] = { 0,1,1,2,3,4,5,6,7,8 };
	iml::copy_order(&temp[1], &temp[2], &temp[10]);
	temp[9] = 9;
	for (size_t i = 0; i < 10; ++i)
		std::cout << temp[i] << std::endl;
		
	return 0;
}
