function

iml::rfind

概要

iml::rfindはイテレータ範囲から任意の要素、もしくは連続した要素の並びの始めの要素のイテレータの位置を逆順で探索する。
また、任意のイテレータ範囲から任意のイテレータ範囲といった連続したデータを探索するとき、探索するデータの数が1であるならば、1つの要素の探索の関数に切り替えるべきである。これは連続データの探索にイテレータの先読みを用いているからであり、イテレータによってはメモリのアクセス違反が起きる。 


namespace iml {
	template <class BidirectionalIterator, class T>
	BidirectionalIterator rfind(BidirectionalIterator first, BidirectionalIterator last, const T& v);
	template <class BidirectionalIterator1, class BidirectionalIterator2>
	BidirectionalIterator1 rfind(BidirectionalIterator1 first1, BidirectionalIterator1 last1, BidirectionalIterator2 first2, BidirectionalIterator2 last2);
}

サンプルコード

入力


#include <IMathLib/IMathLib.hpp>

int main() {

	char* temp = "abcdabfc";
	char* temp2 = "ab";
	std::cout << temp << std::endl;
	std::cout << iml::rfind(temp, &temp[8], 'b') << std::endl;
	std::cout << iml::rfind(temp, &temp[8], temp2, &temp2[2]) << std::endl;

	return 0;
}

出力


abcdabfc
bfc
abcdabfc
