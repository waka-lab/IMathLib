function

iml::fill

概要

iml::fillはイテレータ範囲を任意の値で埋める。
また、戻り値はコピーされた最後のイテレータ位置の次の位置である。 


namespace iml {
	template <class ForwardIterator, class T>
	ForwardIterator fill(ForwardIterator first, ForwardIterator last, const T& v);
}

サンプルコード

入力


#include <IMathLib/IMathLib.hpp>

int main() {

	int temp[10];
	iml::fill(temp, &temp[10], 1);
	for (size_t i = 0; i < 10; ++i)
		std::cout << temp[i] << std::endl;
		
	return 0;
}
