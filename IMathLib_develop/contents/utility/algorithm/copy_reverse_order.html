function

iml::copy_reverse_order

概要

iml::copy_reverse_orderはイテレータの始点から別のイテレータ範囲分を逆順でコピーする。
iml::copy_orderと対称的な関数である。
また、戻り値はコピーされた最後のイテレータ位置の次の位置である 


namespace iml {
	template <class BidirectionalIterator1, class BidirectionalIterator2>
	BidirectionalIterator1 copy_reverse_order(BidirectionalIterator1 first1, BidirectionalIterator2 first2, iml::imsize_t n);
	template <class BidirectionalIterator1, class BidirectionalIterator2>
	BidirectionalIterator1 copy_reverse_order(BidirectionalIterator1 first1, BidirectionalIterator2 first2, BidirectionalIterator2 last2);
}

サンプルコード

入力


#include <IMathLib/IMathLib.hpp>

int main() {

	int temp[10] = { 0,1,5,6,7,8,9,10,11,12 };
	int temp2[3] = { 2,3,4 };
	iml::copy_reverse_order(&temp[5], &temp[2], 5);
	iml::copy_order(&temp[2], temp2, 3);
	for (size_t i = 0; i < 10; ++i)
		std::cout << temp[i] << std::endl;

	return 0;
}
