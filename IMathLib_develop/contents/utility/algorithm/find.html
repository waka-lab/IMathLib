function

iml::find

概要

iml::findはイテレータ範囲から任意の要素、もしくは連続した要素の並びの始めの要素のイテレータの位置を探索する。
また、任意のイテレータ範囲から任意のイテレータ範囲といった連続したデータを探索するとき、探索するデータの数が1であるならば、1つの要素の探索の関数に切り替えるべきである。これは連続データの探索にイテレータの先読みを用いているからであり、イテレータによってはメモリのアクセス違反が起きる。 


namespace iml {
	template <class InputIterator, class T>
	InputIterator find(InputIterator first, InputIterator last, const T& v);
	template <class InputIterator1, class InputIterator2>
	InputIterator1 find(InputIterator1 first1, InputIterator1 last1, InputIterator2 first2, InputIterator2 last2);
}

サンプルコード

入力


#include <IMathLib/IMathLib.hpp>

int main() {

	char* temp = "abcdabfc";
	char* temp2 = "abf";
	std::cout << temp << std::endl;
	std::cout << iml::find(temp, &temp[8], 'd') << std::endl;
	std::cout << iml::find(temp, &temp[8], temp2, &temp2[3]) << std::endl;
		
	return 0;
}

出力


abcdabfc
dabfc
abfc
