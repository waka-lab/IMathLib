struct

iml::type_comparison

概要

iml::type_comparisonは任意の型比較を行う関数の定義されたオブジェクトである。
これは比較演算の抽象化であり、iml::stringやiml::sortで用いられている。 


namespace iml {
	template <class T>
	struct type_comparison {
		using value_type = T;

		static constexpr bool eq(T v1, T v2);
		static constexpr bool lt(T v1, T v2);
	};
	template <>
	struct type_comparison<void> {
		template <class T1, class T2>
		static constexpr auto eq(T1&& v1, T2&& v2);
		template <class T1, class T2>
		static constexpr auto lt(T1 v1, T2 v2);
	};
}

メンバ

関数

名前 説明 対応バージョン
eq 等価比較
lt 小なり比較

型

名前 説明 対応バージョン
value_type オブジェクトの型(T=voidのときは無し)

サンプルコード

入力


#include <IMathLib/IMathLib.hpp>

int main() {
	std::cout << iml::type_comparison<int>::lt(2, 3) << ',' << iml::type_comparison<int>::eq(3, 2) << std::endl;
	return 0;
}

出力


1,0
