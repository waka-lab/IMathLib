struct

iml::object_type

概要

iml::object_typeは任意のデータ構造もしくはデータ型の一番下の階層に存在する値型を取得する。また、その型はtypeに定義される。 


namespace iml {
	template <class T>
	struct object_type;
	template <class T>
	struct object_type<T&>;
	template <class T>
	struct object_type<T&&>;
}

サンプルコード

入力


#include <IMathLib/IMathLib.hpp>

int main() {
	iml::object_type<iml::vector<iml::complex<double>, 5>>::type
		temp = 10;							//double
	std::cout << temp << std::endl;
	return 0;
}

出力


10
