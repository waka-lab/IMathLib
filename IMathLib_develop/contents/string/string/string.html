class

iml::string

概要

iml::stringはあらゆる文字型を使用することができる文字列クラスである。
文字列の連結や取り出し、置換を簡単に実行することができる。なお、原則としてnull文字を代入することは禁止する。また、これはiml::dynamic_containerを継承している。 


namespace iml {
	template <class CharT, class Allocator = iml::allocator<CharT>>
	class string;
	
	using cstring = string<char>;				//char型のstring
	using wstring = string<wchar_t>;			//wchar_t型のstring
}
