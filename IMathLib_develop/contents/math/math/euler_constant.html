function

iml::euler_constant

概要

iml::euler_constantはオイラーの定数を返す。テンプレート引数型を返すことになっているが、整数型の場合には浮動小数点型を返す。 


namespace iml {
	template <class T>
	T euler_constant();
}

実装

オイラーの定数は以下のような定義である。
\[\begin{align*} \gamma=\lim_{n\to\infty}{\left(\sum_{k=1}^{n}\frac{1}{k}-\log{n}\right)} \end{align*}\] しかし、調和級数は対数速度での発散のため非常に計算が遅い。そこで、ディガンマ関数\(\psi(x)\)の以下の関係式を用いることで収束性を改善している。
\[\begin{align*} \psi(1)=-\gamma \end{align*}\]
