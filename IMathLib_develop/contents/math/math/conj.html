function

iml::conj

概要

iml::conjは引数の共役を返す。 


namespace iml {
	template <class T>
	T conj(const T& x);
	template <class T>
	iml::complex<T>& conj(const iml::complex<T>& x);
	template <class T>
	iml::quaternion<T>& conj(const iml::quaternion<T>& x);
	template <class T>
	iml::octonion<T>& conj(const iml::octonion<T>& x);
}

実装

この関数の定義は引数\(x\)が実数であるならば、そのままの値を返す。
実数体上で定義される多元体においては、実数を基底に取る要素以外に対して加法逆元をとったものを返す。
