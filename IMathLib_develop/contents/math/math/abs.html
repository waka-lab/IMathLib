function

iml::abs

概要

iml::absは引数の絶対値を返す。 


namespace iml {
	template <class T>
	T abs(const T& x);
	template <class T>
	T abs(const iml::complex<T>& x);
	template <class T>
	T abs(const iml::vector<T>& x);
}

実装

この関数の定義は引数\(x\)に対して\(\sqrt{xx^{\ast}}\)である。なお、\(x^{\ast}\)は\(x\)の共役もしくは転置である。
iml::vector型に関しては\(L^{2}\)ノルムとして取り扱うことができる。
