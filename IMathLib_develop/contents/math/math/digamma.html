function

iml::digamma

概要

iml::digammaはディガンマ関数を返す。 


namespace iml {
	template <class T>
	T digamma(const T& x);
	template <class T>
	iml::complex<T> digamma(const iml::complex<T>& x);
}

実装

この関数は引数\(z\in{\mathbb C}\)に対して、ディガンマ関数\(\psi(z)\)は \[ \psi(z)=\frac{d}{dz}\log{\Gamma(z)} \] と定義される。そのため、これはiml::lgammaの級数を微分したものとして定義される、すなわち \[ \psi(z)=-\frac{1}{2z}+\log{z}-\sum_{k=1}^{\infty}\frac{B_{2k}}{2kz^{2k}} \] となる。
