function

iml::gamma

概要

iml::gammaはガンマ関数を返す。 


namespace iml {
	template <class T>
	T gamma(const T& x);
	template <class T>
	iml::complex<T> gamma(const iml::complex<T>& x);
}

実装

この関数は引数\(z\in{\mathbb C}\)に対して、以下の積分で定義される。
\[\begin{align*} \Gamma(z)=\int_{0}^{\infty}t^{z-1}e^{-t}dt \end{align*}\] しかし、積分形では計算が難しいため、以下のように計算をする。 \[\begin{align*} \Gamma(z)=e^{\log\Gamma(z)} \end{align*}\] また、\(z\in{\mathbb R}\)としたとき\(\log{\Gamma(z)}\)は\(z>0\)で定義されるため、以下の相反公式を用いることでこの問題を解消している。
\[\begin{align*} \Gamma(z)=\frac{\pi}{\sin{\pi z}}\frac{1}{\Gamma(1-z)} \end{align*}\]
