function

iml::gamma2

概要

iml::gamma2は第二種不完全ガンマ関数を返す。 


namespace iml {
	template <class T>
	T gamma2(const T& a, const T& x);
	template <class T>
	iml::complex<T> gamma2(const iml::complex<T>& a, const T& x);
}

実装

この関数は以下の積分によって定義される。
\[ \begin{align*} \Gamma(a,x)=\int_{x}^{\infty}t^{a-1}e^{-t}dt \end{align*} \] これを\(x\)のまわりでテイラー展開をして変形をすることで以下の式が導かれる。 \[ \begin{align*} \Gamma(a,x)=e^{-x}x^{a}\sum_{k=0}^{\infty}\frac{\prod_{n=1}^{k}(n-a)}{(1+k)!L_{k}^{-a}(-x)L_{1+k}^{-a}(-x)} \end{align*} \] この級数は絶対収束であるが、\(a\)より\(x\)の方が小さいとき収束が遅くなる。このときは\(\Gamma(a)=\gamma(a,x)+\Gamma(a,x)\)の関係式を用いて第一種不完全ガンマ関数に切り替えている。
また、\(L_{k}^{-a}(-x)\)は一般化されたラゲール陪多項式である。
