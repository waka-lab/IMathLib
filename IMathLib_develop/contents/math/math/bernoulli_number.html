function

iml::bernoulli_number

概要

iml::bernoulli_numberはベルヌーイ数を返す。 


namespace iml {
	template <class T>
	T bernoulli_number(iml::imsize_t n);
}

実装

　この関数は引数\(n\)に対して、第二種スターリング数との関係式より \[ B_{n}=\sum_{i=0}^{n}(-1)^{i}i^{n}\sum_{j=i}^{n}\frac{1}{j+1}\binom{j}{i} \] となることを用いている。
　ただし、\(n\)が大きいときは計算オーダーが冪関数的に増大するためiml::bernoulli_number_tableを用いることも検討するべきである。
