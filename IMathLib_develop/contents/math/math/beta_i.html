function

iml::beta_i

概要

iml::beta_iは正規化ベータ関数を返す。 


namespace iml {
	template <class T>
	T beta_i(const T& a, const T& b, const T& x);
	template <class T>
	iml::complex<T> beta_i(const iml::complex<T>& a, const iml::complex<T>& b, const T& x);
}

実装

　この関数はベータ関数と不完全ベータ関数を用いて \[ I_{x}(a,b)=\frac{B_{x}(a,b)}{B(a,b)} \] と定義される。また、\(x>0.5\)であるときは \[ I_{x}(a,b)+I_{1-x}(b,a)=1 \] を用いて計算することで計算効率を上げている。

規約

　ベータ関数および不完全ベータ関数の定義より\(a,b> 0\)および\(0\leq|x|\leq 1\)で動作が保証される。
