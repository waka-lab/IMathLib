function

iml::gamma1

概要

iml::gamma1は第一種不完全ガンマ関数を返す。 


namespace iml {
	template <class T>
	T gamma1(const T& a, const T& x);
	template <class T>
	iml::complex<T> gamma1(const iml::complex<T>& a, const T& x);
}

実装

　この関数は \[ \gamma(a,x)=\int_{0}^{x}t^{a-1}e^{-t}dt \] と定義される。これを\(x\)のまわりでテイラー展開をすることで、 \[ \gamma(a,x)=e^{-x}x^{a}\sum_{k=0}^{\infty}\frac{x^{k}}{\prod_{n=0}^{k}(a+n)} \] とあらわされる。この級数は\(a=0\)の場合を除き絶対収束するが、\(a\)より\(x\)の方が大きいとき収束が遅くなる。このときは\(\Gamma(a)=\gamma(a,x)+\Gamma(a,x)\)の関係式を用いて第二種不完全ガンマ関数に切り替えている。

規約

　第一種不完全ガンマ関数の定義より\(0\leq x\)および\(0<\Re{a}\)で動作が保証される。
