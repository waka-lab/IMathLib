function

iml::sqrt

概要

iml::sqrtは引数の平方根を返す。 


namespace iml {
	template <class T>
	T sqrt(const T& x);
	template <class T>
	T sqrt(const iml::complex<T>& x);
}

実装

　この関数は引数\(x\)の逆数\(y\)に対して \[ \begin{cases} h=1-ay_{n}^{2} \\ y_{n+1}=y_{n}\left(1+\frac{1}{128}\left(64h+h^{2}\left(48+40h+35h^{2}\right)\right)\right) \end{cases} \] といったニュートン法をベースにしたアルゴリズムを用いており、初期値はiml::isqrtにより与えている。
　なお、複素数に対しては\(r,\theta\in {\mathbb R}\)で\(x=re^{i\theta}\)とあらわすことにより \[ \begin{align*} \sqrt{x}=\sqrt{r}e^{i\frac{\theta}{2}} \end{align*} \] と与えられる。
