function

iml::lgamma

概要

iml::lgammaはガンマ関数の自然対数を返す。 


namespace iml {
	template <class T>
	T lgamma(const T& x);
	template <class T>
	iml::complex<T> lgamma(const iml::complex<T>& x);
}

実装

この関数は引数\(z\in{\mathbb C}\)に対して \[ \begin{align*} \log{\Gamma(z)}=\frac{\log{2\pi}}{2}-z+\left(z-\frac{1}{2}\right)\log{z}+\sum_{k=1}^{\infty}\frac{B_{2k}}{2k(2k-1)z^{2k-1}} \end{align*} \] といった漸近展開の式を用いて計算している。また、\(B_{2k}\)はベルヌーイ数である。
このとき、\(|z|\)は十分に大きい数である必要があるが、\(n\in{\mathbb N}\)で \[ \Gamma(z+n)=\prod_{k=0}^{n-1}(z+k)\Gamma(z) \ \Leftrightarrow\ \log{\Gamma(z)}=\log{\Gamma(z+n)}-\log{\left(\prod_{k=0}^{n-1}(z+k)\right)} \] となることを用いることでこの問題を解消している。
