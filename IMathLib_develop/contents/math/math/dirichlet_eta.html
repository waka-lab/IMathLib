function

iml::dirichlet_eta

概要

iml::dirichlet_etaはディリクレのイータ関数を返す。 


namespace iml {
	template <class T>
	T dirichlet_eta(const T& x);
	template <class T>
	iml::complex<T> dirichlet_eta(const iml::complex<T>& x);
}

実装

この関数は引数\(z\in{\mathbb C}\)に対して、以下の交代級数で定義される。
\[\begin{align*} \eta(s)=\sum_{k=1}^{\infty}\frac{(-1)^{k-1}}{k^{s}} \end{align*}\] しかし、この級数は対数収束のため非常に遅い。そこで、Konrad Knoppによるオイラー変換を用いて解析接続することによって以下の級数で定義している。
\[\begin{align*} \eta(s)=\sum_{n=0}^{\infty}\frac{1}{2^{n+1}}\sum_{k=0}^{n}\frac{(-1)^{k}}{(k+1)^{s}}\binom{n}{k} \end{align*}\]
