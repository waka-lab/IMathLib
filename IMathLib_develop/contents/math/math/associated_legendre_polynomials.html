function

iml::associated_legendre_polynomials

概要

　iml::associated_legendre_polynomialsはルジャンドル陪多項式を返す。 


namespace iml {
	template <class T>
	T associated_legendre_polynomials(iml::imsize_t k, iml::imsize_t m, const T& x);
}

実装

　ルジャンドル陪多項式とはルジャンドル多項式\(P_{k}(x)\)に対して \[ P_{k}^{m}(x)=(1-x^{2})^{\frac{m}{2}}\frac{d^{m}}{dx^{m}}P_{k}(x) \] と定義される。この関数は \[ P_{k}^{m}(x)=\frac{(2k-1)xP_{k-1}^{m}(x)-(k+m-1)P_{n-2}^{m}(x)}{k-m},\ P_{m-1}^{m}(x)=0,\ P_{m}^{m}(x)=(2m-1)!!(1-x^{2})^{\frac{m}{2}} \] といった漸化式を用いて計算している。

規約

　ルジャンドル陪多項式の定義より\(m\leq k\)および\(|x|\leq 1\)で動作が保証される。
