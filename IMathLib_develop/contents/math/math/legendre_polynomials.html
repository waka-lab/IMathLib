function

iml::legendre_polynomials

概要

　iml::legendre_polynomialsはルジャンドル多項式を返す。 


namespace iml {
	template <class T>
	T legendre_polynomials(iml::imsize_t k, const T& x);
}

実装

　ルジャンドル多項式とはルジャンドルの微分方程式 \[ \frac{d}{dx}\left((1-x^{2})\frac{dy}{dx}\right)+k(k+1)y=0 \] の一般解に対して初期条件\(y(1)=1\)の下で得られる多項式\(P_{k}(x)\)のことである。この関数は \[ P_{k}(x)=\frac{(2k-1)xP_{k-1}(x)-(k-1)P_{k-2}(x)}{k},\ P_{0}(x)=1,\ P_{1}(x)=x \] といった漸化式を用いて計算している。

規約

　ルジャンドル多項式の定義より\(|x|\leq 1\)で動作が保証される。
