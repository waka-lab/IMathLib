function

iml::isqrt

概要

iml::isqrtは引数の平方根の近似整数を返す。その関数の特性上、実数型においてのみ定義される。 


namespace iml {
	template <class T>
	T isqrt(const T& x);
}

実装

この関数は引数\(x\)に対して、その平方根の近似整数\(n\)で以下の関係式が成り立つことを用いている。 \[\begin{align*} n^{2} = \sum_{k=1}^{n}(2k-1) < x < (n+1)^{2} = \sum_{k=1}^{n+1}(2k-1) \end{align*}\] この関係式で左辺から右辺を引いていくと、以下のようになる。 \[\begin{align*} 0 < &x - \sum_{k=1}^{n}(2k-1) < 2n+1 \\ 0 < &\frac{x}{2} - \sum_{k=1}^{n}k+\frac{n}{2} < n+\frac{1}{2} \\ \end{align*}\] これより、\(x\)に対して自然数を引いていき、\(0\)を下回らない最大の\(n\)が平方根の整数近似となる。
