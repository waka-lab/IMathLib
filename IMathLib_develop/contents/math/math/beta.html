function

iml::beta

概要

iml::betaはベータ関数もしくは不完全ベータ関数を返す。 


namespace iml {
	template <class T>
	T beta(const T& a, const T& b);
	template <class T>
	T beta(const T& a, const T& b, const T& x);
	template <class T>
	iml::complex<T> beta(const iml::complex<T>& a, const iml::complex<T>& b);
	template <class T>
	iml::complex<T> beta(const iml::complex<T>& a, const iml::complex<T>& b, const T& x);
}

実装

　ベータ関数は \[ B(a,b)=\int_{0}^{1}t^{a-1}(1-x)^{b-1}dt \] と定義される。積分形式は計算が困難なため、ガンマ関数との関係式により \[ B(a,b)=\frac{\Gamma(a)\Gamma(b)}{\Gamma(a+b)} \] とあらわされ、自然に定義域が拡張される。
　また、不完全ベータ関数は \[ B_{x}(a,b)=\int_{0}^{x}t^{a-1}(1-x)^{b-1}dt \] と定義され、このとき\(x\in{\mathbb R}\)で\(0\lt x\lt 1\)である必要がある。
　これをテイラー展開することにより \[ B_{x}(a,b)=x^{a}\sum_{k=0}^{\infty}\frac{(-1)^{k}\prod_{n=1}^{k}(b-n)}{k!(k+a)}x^{k} \] とあらわされ、\(|x|<1\)で収束する。
　\(x\)が\(1\)に十分近いときは収束が遅くなるが、 \[ B(a,b)=B_{x}(a,b)+B_{1-x}(b,a) \] を用いることで収束性の改善をしている。

規約

　ベータ関数および不完全ベータ関数の定義より\(a,b> 0\)および\(0\leq|x|\leq 1\)で動作が保証される。
