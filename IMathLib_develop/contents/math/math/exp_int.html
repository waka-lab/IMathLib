function

iml::exp_int

概要

iml::exp_intは引数の指数積分を返す。 


namespace iml {
	template <class T>
	T exp_int(const T& x);
	template <class T>
	iml::complex<T>& exp_int(const iml::complex<T>& x);
	template <class T>
	iml::complex<T>& exp_int(const iml::complex<T>& x, iml::imint_t n);
}

実装

　この関数は引数\(z\in{\mathbb C}\)に対して \[ {\rm Ei(z)}=-PV\int_{-z}^{\infty}\frac{e^{-t}}{t}dt \] と定義される。このとき、オイラーの定数\(\gamma\)を用いて \[ {\rm Ei(-z)}=\gamma+\log{z}+\sum_{k=0}^{\infty}\frac{(-z)^{k}}{kk!} \] が成り立ち、対数の多価性については\(n\in{\mathbb N}\)で \[ {\rm Ei(-z)}=\gamma+{\rm Log\ }{z}+\sum_{k=0}^{\infty}\frac{(-z)^{k}}{kk!}+i2\pi n \] とあらわしている。
　なお、\(z\in{\mathbb R}\)の関数では虚数積分を無視して \[ {\rm Ei(z)}=\gamma+\log{|z|}+\sum_{k=0}^{\infty}\frac{z^{k}}{kk!} \] という実装をしている。

規約

　指数積分の定義より\(x\neq 0\)で動作が保証される。
