function

iml::riemann_zeta

概要

iml::riemann_zetaはリーマンゼータ関数を返す。 


namespace iml {
	template <class T>
	T riemann_zeta(const T& x);
	template <class T>
	iml::complex<T> riemann_zeta(const iml::complex<T>& x);
}

実装

この関数は引数\(z\in{\mathbb C}\)に対して、以下の級数で定義される。
\[\begin{align*} \zeta(s)=\sum_{k=1}^{\infty}\frac{1}{k^{s}} \end{align*}\] しかし、この級数は対数収束のため非常に遅い。そこで、ディリクレのイータ関数\(\eta(s)\)について、\(\eta(s)=(1-2^{1-s})\zeta(s)\)の関係式が成り立つことを用いている。
