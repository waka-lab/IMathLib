Iroha Math Library for C++ リファレンス [ホーム]

header

<IMathLib/interface/interface.hpp>

概要

<IMathLib/interface/interface.hpp>ヘッダではインターフェース制御の統括管理やウィンドウ操作等を提供している。このライブラリでグラフィカルなソフトフェアを作るとき、これがなければ無理である。

ライブラリのフラグセットおよびその周辺

マクロ

名前 説明 対応バージョン
_IMATH_LOG_OUTPUT_ ログファイルの出力をするかの設定
_IMATH_INTERFACE_ インターフェース機能の全てを利用する
_IMATH_INTERFACE_INPUT_ 入力制御を利用する
_IMATH_INTERFACE_AUDIO_ オーディオ制御を利用する
_IMATH_INTERFACE_IMAGE_ テクスチャ制御を利用する
_IMATH_INTERFACE_FONT_ Type True Fontを利用する
_IMATH_INTERFACE_SE_NAM_ SEの数を設定(デフォルトは2)
_IMATH_INTERFACE_CHANNEL_NUM_ SEのチャネルの数を設定(デフォルトは16)
_IMATH_INTERFACE_Z_BUFFER_BITS_ デプスバッファの深度値(デフォルトは24)

構造体

名前 説明 対応バージョン
iml::i_f::rect 矩形領域を示す型

クラス

名前 説明 対応バージョン
iml::i_f::interface_control インターフェース制御の統括管理
iml::i_f::window 1つのウィンドウを示すクラス

関数

名前 説明 対応バージョン
iml::i_f::window_event iml::i_f::windowにより生成されたウィンドウの状態変化のイベントを実行

サンプルコード

以下のサンプルコードはこのヘッダを用いることで可能となる動作であり、このヘッダで提供する関数等のサンプルコードは、これを少し書き換えたものとなる。

入力


#define _IMATH_OPENGL_AND_SDL2_
#include <IMathLib/IMathLib.hpp>
#include <cstdlib>

int main(int argc, char* argv[]) {

	//初期化
	iml::i_f::interface_control::inst()->init();
	//ウィンドウの生成
	iml::i_f::window wnd("main window", 800, 600);
	iml::i_f::window wnd2("sub window", 400, 300);
	//メインウィンドウのクローズイベントのコールバックの設定
	wnd.close_ev([]() {std::exit(0); });
	//ウィンドウの表示
	wnd.show();
	wnd2.show();

	//プログラムのメインループ
	while (1) {
		//イベント処理はSDL2の機能をそのまま呼び出す
		SDL_Event ev;
		while (SDL_PollEvent(&ev)) {
			switch (ev.type) {
			case SDL_WINDOWEVENT:
				iml::i_f::window_event(ev.window);
				break;
			}
		}
	}
	
	return 0;
}

出力

2つのウィンドウが表示され、main windowを閉じるとプログラムが終了する。
