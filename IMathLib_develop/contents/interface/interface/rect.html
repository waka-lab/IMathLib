iml::i_f::rect

概要

iml::i_f::rectは矩形領域を取り扱うための構造体であり、ビューボート領域の選択やカメラの設定等で用いる。
また、デフォルトのコンストラクタでは全てが0となる。 


namespace iml {
	namespace iml {
		template <class T>
		struct rect {
			T left, right, bottom, top;
			constexpr rect();
			constexpr rect(const T& l, const T& r, const T& b, const T& t);
		};
	}
}
